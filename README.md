## üê≥ Emby Docker Server Installation

- This setup is tested only with the `emby/embyserver:beta` Docker image. Other images may require manual testing.
- The official Emby `latest` version (4.8+) has been confirmed to support this script.
- SSH into your server as `root` and run the following command to install:

```bash
wget -O script.sh --no-check-certificate https://raw.githubusercontent.com/jackloves111/Emby.CustomCssJS/main/src/script.sh && bash script.sh
```

- After installation, restart the container. A new plugin for custom JS and CSS will appear in the Emby web console.
- If the plugin doesn‚Äôt show up, check the permissions of the mapped `config` folder.
